# SWAN Shipping Corporation

Modern web application for SWAN Shipping, a leading LPG maritime transport provider.

## Features

-  Fleet showcase with 19+ LPG carriers
-  News and company updates
-  Career opportunities portal
-  Multi-form contact system
-  Fully responsive design

## Tech Stack

- React 19
- Vite 7
- Tailwind CSS 4.1
- React Router 7

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── features/        # Feature modules (fleet, careers, news, contact, legal)
├── components/      # Shared UI components
├── config/          # App configuration
├── utils/           # Utility functions
├── hooks/           # Custom React hooks
├── services/        # API service layer
└── App.jsx          # Main application component
```
